1. In Jacobson, this word is pointed וְלָרֻאוּבֵנִ֨י, i.e. with a qubuts under the resh. In my opinion this is excessive adherence to ל because ל is anomalous here. No reader-friendly edition will be pointed that way.

2. In this first appearance (page 242), this word appears without an acccent, i.e. as עׇזִּי. Subsequently, it appears with mahapakh.

3. In this first appearance (page 242), this word appears without a locale reference, by design. Subsequently, it appears with locale reference Exod. 15:2.

4. The word חֹ֥דֶשׁ seems like a weird choice since a short not long /o/ would be expected. Seems like a word with qamats qatan would have been a better choice. Maybe קׇדְשִֽׁי׃ or similar?

5. This locale reference (Exod. 20:12) uses BHS verse numbering; in MAM verse numbering, the locale reference is Exod 20:11.

6. In Jacobson, this word is pointed שֶׁשֱּׁזָפַ֖תְנִי, i.e. with a ḥataf segol under the second shin. In my opinion this is excessive adherence to ל because ל is anomalous here. No reader-friendly edition will be pointed that way.

7. In Jacobson, this word is pointed וְהַֽלְוִיִּם֒, i.e. with a meteg on הַ. MAM lacks such a meteg not only in this location (2Chr. 24:5) but in all locations where words ending in לְוִיִּם appear.

8. Jacobson does not provide a locale reference for בִּנְבִיאִ֔ים and I cannot find one. As such, perhaps לִנְבִיאִ֔ים in Amos 2:11 would be a better example?

9. The exact same word (וַיָּ֣קׇם), down to the accent (munaḥ) appears later on the same page (271), and on page 272. But in those later two places, a different locale reference is used: Gen. 22:3. Using a different locale reference for the same word seems pointless, and possibly not harmless: the detail-oriented reader might assume that a different locale reference implies a different word. Such a reader might search for a difference in these two words and be frustrated to find no difference.

10. In Jacobson, this word is pointed מִֽמָּ֣חֳרָ֔ת, i.e. with a munaḥ under the second mem. Jacobson does not provide a locale reference for this word and I cannot find one with this particular munaḥ pointing. In Jacobson this word is pointed with a qamats (ambiguous as to gadol/qatan) under the second mem. This pointing is expected since in Jacobson the qamats mark is always intentionally ambiguous. And anyway here Jacobson is highlighting a dispute about the “quality” (gadol/qatan) of this particular qamats. In the theoretical (ד) variant of MAM, this mem is pointed with qamats qatan. In Jacobson, this word is is discussed in the body text as if it had already been mentioned (e.g. as if it had already appeared in the table above) but it has not already been mentioned.
